# 뭐하려 했더라 — 사용 설명서

> 전화를 받다가, 회의를 하다가 "아 이거 해야 하는데" 싶은 일을
> **일단 던져두면, 잊지 않게 챙겨주는** 개인 업무 보드입니다.
> 인터넷도, 설치도, 계정도 필요 없습니다.
>
> (v2.4.0부터 이름이 '뭐해야 했더라' → **'뭐하려 했더라'** 로 바뀌었습니다.
> 기존 데이터·백업 파일은 그대로 호환됩니다.)

---

## 0. 3분 요약 — 이것만 알면 됩니다

1. **던진다** — 메모를 적고 `Ctrl+Enter`. 다른 프로그램을 쓰는 중이면 `Ctrl+Alt+Space`.
2. **정리한다** — 시간 날 때 '분류 대기' 카드를 눌러 마감·세부할일을 채웁니다.
3. **프로그램이 배치한다** — 카드는 손으로 옮기지 않습니다. 시각을 보고 알아서 칸에 놓입니다.
4. **알람이 챙긴다** — 시각이 되면 창이 앞으로 튀어나옵니다. 끝난 일은 체크 → [완료] 탭.

자주 쓰는 키:

| 키 | 하는 일 |
|---|---|
| `Ctrl+Enter` | 메모 등록 (메인 입력창·빠른 메모창 공통) |
| `Ctrl+Alt+Space` | 어디서든 빠른 메모창 열기 (고정 단축키) |
| `Alt` (빠른 메모창에서) | 메모 ↔ 내 업무 검색 전환 |
| `Esc` | 창 닫기 — 쓰던 내용은 지워지지 않음 |

## 1. 시작하기

1. `뭐하려 했더라.exe` 를 아무 곳(바탕화면, D드라이브 등)에 복사합니다.
2. 더블클릭하면 끝. **설치도, 관리자 권한도 필요 없습니다.**
3. 첫 실행 때 "데이터를 어디에 보관할까요?" 폴더 선택창이 열립니다.
   - 폴더를 고르면 그 안에 `뭐하려했더라_데이터` 폴더가 생기고, 데이터와 자동 백업이 전부 거기에만 저장됩니다.
   - 건너뛰어도 됩니다 — 나중에 [설정] 메뉴의 **[저장 위치]** 로 언제든 옮길 수 있습니다.

이미 켜져 있는데 또 실행하면 새 창 대신 기존 창이 앞으로 나옵니다.

## 2. 보드 — 화면 한눈에 보기

![보드 화면 — 분류 대기·오늘 처리·진행 중·예정·대기 네 칸](docs/screenshots/board.png)

> 예시 데이터 화면입니다. 오른쪽 위에 현재 버전(v2.5.2)이 보입니다.

### 카드 읽는 법

카드 한 장은 **세 줄**이 전부입니다. 각 줄은 한 행으로만 표시됩니다(길면 …).

| 줄 | 내용 |
|---|---|
| 1줄 | **메모** — 무슨 일인지 |
| 2줄 | **세부할일** — 다음에 할 중간 단계 (있을 때만) |
| 3줄 | **시각** — `점검:`(다가오는 세부할일 시각) 또는 `마감:`(마감 시각) |

- 시각 태그는 **임박(지났거나 2시간 이내)일 때만 붉게** 강조됩니다. 그 외엔 중립색.
- `주기` 배지 = 주기 업무에서 자동 생성된 카드.
- 시각 앞의 작은 점(●)은 알람 상태입니다 → [4. 알람](#4-알람--잊지-않게) 참고.
- 관련인·진행상황·담당자·파일 같은 자세한 내용은 **카드를 클릭**해 양식에서 봅니다.

### 칸은 프로그램이 정합니다

**카드를 직접 끌어다 옮길 필요가 없습니다.** 마감·세부 시각(과 담당자)을 보고
알아서 놓아주고, 각 칸 안에서는 **시각이 먼저 도래하는 일이 위**로 올라옵니다.
맨 위 카드부터 처리하면 됩니다.

보드는 **두 가지 방식**으로 볼 수 있습니다. 앱 이름 옆 `시간 │ 시간·담당자` 버튼으로
전환하세요(선택은 저장됩니다).

**① 시간 방식 (기본)** — "언제 할 일인가"로 나눕니다.

| 칸 | 여기에 오는 일 |
|---|---|
| **분류 대기** | 방금 던져둔 메모. 아직 마감을 정하지 않은 일 |
| **오늘 처리** | 오늘이 마감이거나, 오늘 점검할 게 있거나, 이미 지난 일 |
| **진행 중** | 마감은 나중이지만 그 전에 점검할 세부할일이 잡혀 있는 일 |
| **예정 · 대기** | 마감이 내일 이후라 아직 여유가 있는 일 |

**② 시간·담당자 방식** — "내 일인가, 남에게 넘겨 기다리는 일인가"를 함께 봅니다.

![시간·담당자 보드 — 다섯 칸](docs/screenshots/board-owner.png)

| 칸 | 여기에 오는 일 |
|---|---|
| **분류 대기** | 방금 던져둔 메모 (시간 방식과 동일) |
| **본인 진행 · 오늘** | 담당이 나, 오늘 처리할(또는 지난) 일 |
| **타인 진행 · 오늘** | 남에게 넘겨둔 일 중 오늘 챙겨야 하는 일 |
| **본인 진행·오늘 외** | 담당이 나, 오늘 처리할 시각이 없는 일(내일 이후·무시각) |
| **타인 진행·오늘 외** | 남에게 넘겨둔 일 중 오늘 처리할 시각이 없는 일 |

- **담당자는 세부할일별로** 적습니다(비워두면 본인).
- 카드의 본인/타인은 *가장 먼저 도래하는 미완료 세부할일의 담당자*로 판정합니다.
  담당자를 적은 세부할일이 없으면 본인 업무입니다.

## 3. 일 던져두기 (입력)

세 가지 입력 방법이 있습니다. **급할수록 위의 것**을 쓰세요.

### ① 바로 입력 — 가장 빠름
화면 위 입력창에 적고 `Ctrl+Enter` (또는 [바로 입력] 버튼).
'분류 대기'에 쌓입니다. 전화 받는 중이라면 일단 이걸로.

### ② 빠른 메모창 — 다른 프로그램을 쓰는 중일 때
문서·결재 화면을 보다가도 `Ctrl+Alt+Space` → 작은 메모창이 뜹니다.
적고 `Ctrl+Enter` — 끝. '분류 대기'로 들어갑니다.

![빠른 메모창 — 화면 어디서든 떠오르는 입력바](docs/screenshots/capture-memo.png)

믿고 써도 되는 이유:
- `Esc`나 창 전환으로 닫혀도 **쓰던 내용은 남습니다.** 지우는 건 직접 지울 때만.
- 쓰다 만 채 컴퓨터가 꺼져도 **다음 실행 때 '분류 대기'에 자동 등록**됩니다.
- `Alt` 를 누르면 **내 업무 검색**으로 바뀝니다. 결과마다 진행/완료 태그가 붙고,
  클릭하면 메인 창에서 열립니다. `Alt` 를 다시 누르면 메모로 복귀(내용 그대로).

### ③ 양식 입력 — 제대로 정리할 때
[양식 입력] 버튼(또는 카드 클릭)으로 자세한 양식을 엽니다.

| 항목 | 적는 것 |
|---|---|
| **메모** | 일의 내용 |
| **관련소속 · 관련인 · 연락처** | 누가 요청했는지 (여러 명 가능) |
| **접수 · 마감시각** | 받은 때와 최종 기한 |
| **식별번호** | 입찰공고번호, SR번호 같은 번호들 |
| **세부 할일** | 중간 단계들 — 단계마다 점검 시각·**담당자**(비우면 본인) 지정 가능 |
| **파일 링크** | 관련 문서(한글·엑셀 등) 연결 |

![양식 입력](docs/screenshots/form.png)

- **파일 링크**는 행마다 켜고 끌 수 있습니다:
  **켜면** 파일 이름이 링크가 되어 클릭 즉시 열리고(수정 잠금),
  **끄면** 경로를 직접 고치거나 [찾기]로 파일을 다시 고를 수 있습니다.

![파일 링크 — 켜면 클릭으로 열기, 끄면 경로 수정](docs/screenshots/filelink.png)

- 세부 할일·프리셋·식별번호 명칭은 왼쪽 ⠿ 손잡이를 잡고 **드래그로 순서 변경**.

### 주기 업무 — 반복되는 일은 한 번만 등록
매주 보고, 매월 점검 같은 일은 **[설정] → 주기 업무 입력·관리**에서 등록합니다.

1. 공통 내용(제목/메모)과 반복 주기만 정합니다 —
   `매주 요일`(월·수 등) 또는 `매월 며칠`(그 달에 없는 날짜는 말일로 보정) + 시각.
2. 그 주가 시작되면(월요일 아침) 그 주 회차가 **보드에 자동으로 생깁니다.**
   생긴 업무는 보통 업무와 똑같이 다루고, 카드에 `주기` 배지가 붙습니다.
3. 등록한 주기는 같은 화면에서 수정·일시정지·삭제하고, 다음 예정일도 확인합니다.

![주기 업무 입력·관리](docs/screenshots/recur.png)

> 주기(부모)는 보드에 카드로 보이지 않습니다. 실제 할 일만 그때그때 생겨나며,
> 주기를 지워도 이미 생긴 업무는 남습니다.

### 프리셋 — 자주 하는 일은 틀을 저장
"OO 검토 요청"처럼 형태가 반복되는 일은 프리셋으로 저장하세요.
버튼 한 번에 세부할일까지 채워진 양식이 열립니다.
관리는 **[설정] → 프리셋·식별번호 관리**.

![설정 메뉴](docs/screenshots/settings.png)

## 4. 알람 — 잊지 않게

마감·점검 시각이 되면 **알림창이 뜨고 소리가 납니다** (프로그램이 뒤에 있어도 앞으로 나옵니다).

| 버튼 | 하는 일 |
|---|---|
| **[확인]** | 알람 종료 — 카드에 확인 표시(초록 점)가 남음 |
| **[10분 뒤 다시]** | 잠시 미룸 |

카드의 작은 점(●)이 알람 상태입니다: **빈 점**(대기) → **빨강**(울림) → **초록**(확인).
완료한 업무에는 점이 없습니다. 알람 전체는 상단 알람 버튼으로 켜고 끕니다.

## 5. 완료 · 검색 · 캘린더

**완료** — 카드 왼쪽 동그라미를 체크하면 [완료] 탭으로 갑니다.
완료 탭 카드는 메모와 `완료: 시각`을 보여줍니다.
카드를 ×로 지웠다면 직후 알림에서 **실행 취소**할 수 있습니다.

**검색** — 검색창에 아무거나: 메모 내용, 관련인, 소속, 담당자, 번호, 파일명.
연락처는 `01012345678`처럼 숫자만 쳐도 찾아집니다.

**캘린더** — 이번 달 일정을 한눈에. **진행 중인 일만** 나옵니다:
그날 마감(빨강)과 그날 점검할 세부할일(회청색)만 표시되고, 완료된 것은 나오지 않습니다.
날짜를 누르면 그날의 일 목록이 아래에 열립니다.

![캘린더 — 진행 중인 마감·점검만 표시](docs/screenshots/calendar.png)

## 6. 창을 닫아도 꺼지지 않습니다 (트레이)

창의 ×를 눌러도 프로그램은 **화면 오른쪽 아래 트레이(시계 옆)** 에 남아
알람과 빠른 메모 단축키를 계속 챙깁니다.
- 트레이 아이콘 클릭 → 다시 열기
- 트레이 아이콘 우클릭 → 종료

## 7. 내 데이터는 안전한가요?

- **저장은 자동.** 바꾸는 즉시 저장됩니다 — 저장 버튼이 아예 없습니다.
- **자동 백업**: 30분마다 + 날마다 첫 백업본을 14일간 보관.
- 시작할 때 데이터 파일이 손상돼 있으면 **최근 백업으로 스스로 복구**합니다.
- 모든 데이터는 **내가 고른 폴더 안에만** 있습니다. 인터넷으로 나가는 것은 없습니다.

**다른 컴퓨터로 옮기기** — [설정] 메뉴에서:

| 버튼 | 용도 |
|---|---|
| **[JSON파일 백업]** | 데이터 전체를 파일 하나로 → 새 PC에서 [불러오기]로 복원 |
| **[JSON·DB파일 불러오기]** | JSON 백업 또는 .sqlite 백업을 선택해 복원 |
| **[xlsx]** | 엑셀 내보내기 — **보고서용** (복원 불가, 이사·백업은 JSON으로) |

## 8. 자주 묻는 질문

**Q. 인터넷이 안 되는 PC인데 쓸 수 있나요?**
네. 인터넷을 전혀 사용하지 않습니다. 내부망·폐쇄망에서 그대로 동작합니다.

**Q. 설치 권한이 없는데요?**
설치가 없습니다. exe 파일 하나를 복사해 실행하면 끝입니다.

**Q. 실행이 안 돼요.**
구형 Windows 10이라면 "WebView2 런타임"이 없을 수 있습니다(Windows 11·최신
Windows 10은 기본 내장). 전산 담당자에게 WebView2 오프라인 설치를 요청하세요.

**Q. 알람 소리가 안 나요.**
상단 알람 버튼이 꺼져 있지 않은지, Windows 알림 설정에서 차단되지 않았는지 확인하세요.

**Q. 컴퓨터를 바꾸게 됐어요.**
[JSON파일 백업]으로 파일 하나를 만들어 가져간 뒤, 새 PC에서
[JSON·DB파일 불러오기]를 하면 됩니다. exe도 같이 복사하면 끝.

**Q. 카드 위치를 직접 옮기고 싶어요.**
이 보드는 위치를 손으로 옮기지 않는 게 특징입니다 — 시각만 맞게 넣어두면
"오늘 뭘 해야 하는지"를 프로그램이 정리해 줍니다. 옮기고 싶다면 카드를 눌러
마감·점검 시각을 조정하세요.
