# 뭐하려 했더라 — 사용 설명서

> 전화를 받다가, 회의를 하다가 "아 이거 해야 하는데" 싶은 일을
> **일단 던져두면, 잊지 않게 챙겨주는** 개인 업무 보드입니다.
> 인터넷도, 설치도, 계정도 필요 없습니다.
>
> (v2.4.0부터 프로그램 이름이 '뭐해야 했더라'에서 **'뭐하려 했더라'** 로
> 바뀌었습니다. 기존 데이터·백업 파일은 그대로 호환됩니다.)

---

## 1. 시작하기

1. `뭐하려 했더라.exe` 파일을 컴퓨터 아무 곳(바탕화면, D드라이브 등)에 복사합니다.
2. 더블클릭하면 바로 실행됩니다. **설치 과정도, 관리자 권한도 필요 없습니다.**
3. 처음 실행하면 "데이터를 어디에 보관할까요?" 폴더 선택창이 열립니다.
   원하는 폴더를 고르면 그 안에 `뭐하려했더라_데이터` 폴더가 생기고,
   내 업무 데이터와 자동 백업이 전부 그 안에만 저장됩니다.
   (건너뛰어도 됩니다 — 나중에 상단 **[저장 위치]** 버튼으로 언제든 바꿀 수 있습니다.)

이미 켜져 있는데 또 실행하면 새 창이 뜨는 게 아니라 기존 창이 앞으로 나옵니다.

---

## 2. 화면 한눈에 보기 — 네 개의 칸

![보드 화면 — 분류 대기·오늘 처리·진행 중·예정·대기 네 칸](docs/screenshots/board.png)

> 위 화면은 예시 데이터입니다. 오른쪽 위에 현재 버전(v2.4.3), 카드의 `주기` 배지는
> 주기 업무에서 자동 생성된 업무를 뜻합니다.

보드에는 칸이 네 개 있습니다. **카드를 직접 끌어다 옮길 필요가 없습니다** —
마감시각과 세부할일 시각을 보고 프로그램이 알아서 놓아줍니다.

| 칸 | 여기에 오는 일 |
|---|---|
| **분류 대기** | 방금 던져둔 메모. 아직 마감을 정하지 않은 일 |
| **오늘 처리** | 오늘이 마감이거나, 오늘 점검할 게 있거나, 이미 지난 일 |
| **진행 중** | 마감은 나중이지만 그 전에 점검할 세부할일이 잡혀 있는 일 |
| **예정 · 대기** | 마감이 내일 이후라 아직 여유가 있는 일 |

각 칸 안에서는 **시각이 먼저 도래하는 일이 위**로 올라옵니다.
맨 위의 카드부터 처리하면 됩니다.

## 3. 일 던져두기 (입력)

### 바로 입력 — 가장 빠른 방법
화면 위 입력창에 메모를 적고 **Ctrl+Enter** (또는 [바로 입력] 버튼).
"분류 대기" 칸에 쌓입니다. 전화 받는 중이라면 일단 이걸로 던져두세요.

### 메모창 단축키 — 다른 프로그램을 쓰는 중일 때
다른 창(문서, 결재 화면 등)을 보고 있어도 **Ctrl+Alt+Space** 를 누르면
작은 메모창이 즉시 뜹니다. 적고 **Ctrl+Enter** — 끝. "분류 대기"에 자동으로 들어갑니다.
(단축키는 Ctrl+Alt+Space 고정입니다.)

![빠른 메모창 — 화면 어디서든 떠오르는 스포트라이트식 입력바](docs/screenshots/capture-memo.png)

메모창의 든든한 점:
- **Esc나 창 전환으로 닫혀도 쓰던 내용은 그대로 남습니다.** 지우는 건 직접 지울 때만.
- 쓰다 만 메모가 있는 채로 컴퓨터가 꺼져도, **다음 실행 때 그 내용이 '분류 대기'에
  자동 등록**됩니다 — 전화 중 급히 끄적인 메모가 사라지지 않습니다.
- **Alt 키**를 누르면 검색 모드로 바뀝니다: 내 업무가 즉시 검색되고, 업무를 클릭하면
  메인 창에서 열립니다. Alt를 다시 누르면 메모 모드로 돌아옵니다(쓰던 내용 그대로).

### 양식 입력 — 제대로 정리할 때
[양식 입력] 버튼을 누르면 자세한 양식이 열립니다:
- **관련인**: 누가 요청했는지 (이름 · 소속 · 연락처)
- **식별번호**: 입찰공고번호, SR번호 같은 번호들
- **세부 할일**: 이 일을 쪼갠 중간 단계들 — 각각 점검 시각을 걸 수 있습니다
- **파일 링크**: 이 일과 관련된 문서(한글, 엑셀 등)를 연결합니다.
  **켜면(활성화)** 파일 이름이 링크로 나오고 **클릭하면 바로 열립니다**(수정 잠금),
  **끄면(비활성화)** 경로 입력칸으로 바뀌어 직접 수정하거나 옆의 **[찾기]** 버튼으로 파일을 다시 고를 수 있습니다.
- **마감시각**: 최종 기한

![양식 입력 — 메모·관련인·접수/마감시각·식별번호·세부 할일·파일 링크](docs/screenshots/form.png)

![파일 링크 — 켜면 이름 클릭으로 열기(수정 잠금), 끄면 경로 수정·[찾기]로 다시 선택](docs/screenshots/filelink.png)

세부 할일·프리셋·식별번호 명칭은 왼쪽의 ⠿ 손잡이를 잡고 **드래그로 순서를
바꿀 수 있습니다.** 카드를 클릭하면 언제든 이 양식이 다시 열려 수정할 수 있습니다.

### 주기 업무 — 반복되는 일은 한 번만 등록 (v2.4.0)
매주 하는 보고, 매월 정기 점검 같은 일은 **[설정] 메뉴 → 주기 업무 입력·관리**
에서 등록합니다. 일반 양식 입력과는 별개예요.
1. **공통 내용(제목/메모)** 과 **반복 주기**만 정합니다 — `매주 요일`(월·수 등 선택) /
   `매월 며칠`(예: 매월 15일 — 그 달에 없는 날짜는 말일로 맞춰집니다) + 시각.
2. 예정일이 되면 그 내용으로 **업무가 보드에 자동으로 생깁니다**(자식 업무).
   자식 업무는 보통 업무와 똑같이 관련인·세부 할일 등을 채우고, 완료하면 [완료] 탭으로 갑니다.
   카드에는 `주기` 배지가 붙습니다.
3. 등록한 주기(부모)는 **[설정] 메뉴 → 주기 업무 입력·관리** 에서 목록으로 보고
   수정·일시정지·삭제할 수 있습니다. 다음 예정일과 지금까지 생성된 개수도 함께 보입니다.

![주기 업무 입력·관리 — 공통 내용과 반복 주기만 등록, 아래에 등록 목록](docs/screenshots/recur.png)

> 부모(주기 정의)는 보드에 카드로 보이지 않습니다 — 공통 내용만 담아두고, 실제로 할 일은
> 자식 업무로 그때그때 생겨납니다. 부모를 지워도 이미 생긴 자식 업무는 그대로 남습니다.

### 프리셋 — 자주 하는 일은 틀을 저장
같은 형태의 업무가 반복된다면(예: "OO 검토 요청") 프리셋으로 저장해 두세요.
버튼 한 번으로 세부할일까지 채워진 양식이 열립니다.
프리셋 관리(만들기·수정·순서 변경)는 상단 **[설정] 메뉴 → 프리셋·식별번호 관리**에 있습니다.

상단 오른쪽 **[설정]** 버튼을 누르면 저장 위치·백업·XLSX·프리셋·주기 업무 관리가 한곳에 모여 있습니다.

![설정 메뉴 — 저장 위치·JSON 백업·불러오기·XLSX·프리셋·주기 업무 입력·관리](docs/screenshots/settings.png)

## 4. 알람 — 잊지 않게

마감시각이나 세부할일 점검 시각이 되면:
- 알림창이 뜨고 소리가 납니다 (프로그램이 뒤에 있어도 앞으로 나옵니다)
- **[확인]** — 알람을 껐다는 표시(초록 점)가 카드에 남습니다
- **[10분 뒤 다시]** — 잠시 미룹니다

카드의 작은 점(●)이 알람 상태입니다: 대기(빈 점) → 울림(빨강) → 확인(초록).
**완료한 업무에는 알람 점이 표시되지 않습니다**(끝난 일이라 알람이 의미 없으니까요).
상단 알람 버튼으로 알람 전체를 켜고 끌 수 있습니다.

## 5. 완료 · 검색 · 캘린더

- **완료**: 카드 왼쪽 체크박스를 누르면 보드에서 떠나 [완료] 탭에 보관됩니다.
  실수로 지웠다면? 삭제 직후 뜨는 알림에서 **실행 취소**를 누르세요.
- **검색**: 검색창에 이름, 번호, 메모 내용, 연결한 파일명 아무거나 — 카드가 바로 걸러집니다.
- **캘린더**: 달력 탭에서 이번 달 일정을 한눈에 봅니다. **진행 중인 일만** 나옵니다 —
  그날이 마감일인 업무(빨강)와 그날 시각이 걸린 세부 할일(회청색)만 표시되고,
  **완료된 업무·완료된 세부 할일은 나타나지 않습니다.** 날짜를 누르면 그날의 일이 나옵니다.

![캘린더 — 그달의 마감일과 세부 할일 시각을 한눈에 (진행 중만 표시)](docs/screenshots/calendar.png)

참고로 저장 위치·JSON 백업·불러오기·XLSX 다운로드 버튼도 전부 [설정] 메뉴 안으로
모였습니다.

## 6. 창을 닫아도 꺼지지 않습니다 (트레이)

창의 X를 눌러도 프로그램은 **화면 오른쪽 아래 트레이(시계 옆)** 에 남아
알람과 메모창 단축키를 계속 챙깁니다. 트레이 아이콘을 누르면 다시 열리고,
완전히 끝내려면 트레이 아이콘에서 종료를 선택하세요.

---

## 7. 내 데이터는 안전한가요?

- **저장은 자동입니다.** 뭔가 바꾸는 즉시 저장됩니다 — 저장 버튼을 찾지 마세요.
- **자동 백업**: 30분마다, 그리고 날마다 첫 백업본을 14일간 따로 보관합니다.
- 시작할 때 데이터 파일이 손상돼 있으면 최근 백업으로 스스로 복구합니다.
- 모든 데이터는 **내가 고른 폴더 안에만** 있습니다. 인터넷으로 나가는 것은 아무것도 없습니다.

### 다른 컴퓨터로 옮기기 / 수동 백업
- **[JSON파일 백업]**: 데이터 전체를 파일 하나로 저장합니다. 새 PC에서
  **[JSON·DB파일 불러오기]** 로 그 파일을 선택하면 그대로 복원됩니다.
- **[xlsx]**: 엑셀 파일로 내보내기 — 보고서용입니다. (엑셀 파일로는 복원이 안 되니,
  이사·백업용은 꼭 JSON을 쓰세요.)

## 8. 자주 묻는 질문

**Q. 인터넷이 안 되는 PC인데 쓸 수 있나요?**
네. 이 프로그램은 인터넷을 전혀 사용하지 않습니다. 내부망·폐쇄망에서 그대로 동작합니다.

**Q. 설치 권한이 없는데요?**
설치가 아예 없습니다. exe 파일 하나를 복사해서 실행하면 끝입니다.

**Q. 실행이 안 돼요.**
Windows 10 구형 PC라면 "WebView2 런타임"이 없을 수 있습니다 (Windows 11과
최신 Windows 10에는 기본 내장). 전산 담당자에게 WebView2 오프라인 설치를 요청하세요.
그 외에는 별도 프로그램이 필요 없습니다.

**Q. 알람 소리가 안 나요.**
상단의 알람 버튼이 꺼져 있지 않은지, Windows 알림 설정에서 차단되지 않았는지 확인하세요.

**Q. 컴퓨터를 바꾸게 됐어요.**
[JSON파일 백업]으로 파일 하나를 만들어 새 PC로 가져간 뒤, 새 PC에서
[JSON·DB파일 불러오기]를 하면 됩니다. exe 파일도 같이 복사하면 끝.

**Q. 카드 위치를 직접 옮기고 싶어요.**
이 보드는 위치를 직접 옮기지 않는 게 특징입니다 — 마감·점검 시각만 맞게 넣어두면
"오늘 뭘 해야 하는지"를 프로그램이 대신 정리해 줍니다. 위치를 바꾸고 싶다면
카드를 눌러 마감시각을 조정하세요.
